Portfolio Details

Web application for account management

This banking management web application is designed to simplify and automate the management of accounts and banking operations. It offers an intuitive interface and a comprehensive set of features for users and administrators.

Key Features

  1. Secure authentication :
    • Login : Secure access for existing users via entering credentials.
    • Signup : Registration of new users with validation of information.
  2. Account management :
    • Consultation of accounts: Each user can consult their bank accounts, view the balances and details of each account.
    • List of operations: Detailed display of transactions (withdrawals, deposits, transfers) carried out on each account, with the possibility of filtering by type or date.
  3. Data export :
    • PDF generation: Users can download a PDF report of transactions related to a specific account, with a structured design including information such as generation date and page number.
    • Export to Excel: In addition to PDF, the application also allows you to generate an Excel file with the same information.
  4. Banking operations :
    • Carry out an operation: The user can carry out banking operations such as:
      • Withdrawal: Subtraction of an amount from an account.
      • Disbursement: Adding funds to an account.
      • Transfer: Transfer of money between two different accounts.
  5. Customer management :
    • Adding a New Customer: Administrators can register new customers in the system with detailed information (name, address, phone number, etc.).

Technologies used

  • Backend : Spring Boot for managing REST APIs, securing with JWT for authentication.
  • Frontend : React for an interactive and responsive user interface.
  • Database: Storage of banking and customer information via a relational database (JPA/Hibernate).
  • Data export: Use of libraries for generating PDF and Excel files.

This application offers a complete and reliable solution for simplified management of banking operations, accessible to users while offering efficient administration tools for managers.
