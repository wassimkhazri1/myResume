Portfolio Details

Application de gestion des finances personnelles

Budget Keeper est une application web complète conçue pour aider les utilisateurs à suivre leurs dépenses, gérer leur budget et analyser leurs habitudes de dépenses. Elle offre une interface conviviale avec authentification sécurisée et visualisation des données en temps réel.

Fonctionnalités Clés

  1. Authentification sécurisée :
    • Connexion : Accès sécurisé pour les utilisateurs existants via la saisie des identifiants.
    • Inscription : Inscription des nouveaux utilisateurs avec validation des informations.
  2. Suivi des dépenses :
    • Ajouter/Modifier/Supprimer des transactions
    • Catégoriser les dépenses (nourriture, factures, etc.)
  3. Gestion budgétaire
    • Fixer des limites budgétaires mensuelles
    • Suivi visuel des progrès
  4. Rapports
    • Graphiques interactifs (React Charts.js)

Technologies Utilisées

  • Backend: Spring Boot, Spring Security, JPA/Hibernate
  • Frontend: React, Bootstrap
  • Base de données: PostgreSQL
  • Authentification: JWT

Un outil de suivi des dépenses convivial avec gestion budgétaire, aidant les particuliers à surveiller leurs dépenses et à maintenir leur discipline financière.
