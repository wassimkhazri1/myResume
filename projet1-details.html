Application web pour la gestion des comptes

Cette application web de gestion bancaire est conçue pour simplifier et automatiser la gestion des comptes et des opérations bancaires. Elle offre une interface intuitive et un ensemble complet de fonctionnalités pour les utilisateurs et les administrateurs.

Fonctionnalités clés

  1. Authentification sécurisée :
    • Login : Accès sécurisé pour les utilisateurs existants via la saisie d’identifiants.
    • Signup : Inscription de nouveaux utilisateurs avec validation des informations.
  2. Gestion des comptes :
    • Consultation des comptes : Chaque utilisateur peut consulter ses comptes bancaires, visualiser les soldes et les détails de chaque compte.
    • Liste des opérations : Affichage détaillé des transactions (retraits, versements, transferts) effectuées sur chaque compte, avec la possibilité de filtrer par type ou date.
  3. Exportation des données :
    • Génération de fichiers PDF : Les utilisateurs peuvent télécharger un rapport PDF des opérations liées à un compte spécifique, avec un design structuré comprenant des informations comme la date de génération et le numéro de page.
    • Exportation en Excel : En plus du PDF, l’application permet également de générer un fichier Excel avec les mêmes informations.
  4. Opérations bancaires :
    • Effectuer une opération : L'utilisateur peut réaliser des opérations bancaires telles que :
      • Retrait : Soustraction d’un montant d’un compte.
      • Versement : Ajout de fonds à un compte.
      • Transfert : Transfert d’argent entre deux comptes différents.
  5. Gestion des clients :
    • Ajout d'un nouveau client : Les administrateurs peuvent enregistrer de nouveaux clients dans le système avec des informations détaillées (nom, adresse, numéro de téléphone, etc.).

Technologies utilisées

  • Backend : Spring Boot pour la gestion des API REST, sécurisation avec JWT pour l'authentification.
  • Frontend : React pour une interface utilisateur interactive et responsive.
  • Base de données : Stockage des informations bancaires et des clients via une base de données relationnelle (JPA/Hibernate).
  • Exportation des données : Utilisation de bibliothèques pour la génération de fichiers PDF et Excel.

Cette application offre une solution complète et fiable pour une gestion simplifiée des opérations bancaires, accessible aux utilisateurs tout en offrant des outils d'administration performants pour les gestionnaires.
