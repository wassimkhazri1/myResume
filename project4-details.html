Portfolio Details

Personal Finance Management App

Budget Keeper is a full-stack web application designed to help users track expenses, manage budgets, and analyze spending habits. It provides a user-friendly interface with secure authentication and real-time data visualization.

Key Features

  1. Secure authentication :
    • Login : Secure access for existing users via entering credentials.
    • Signup : Registration of new users with validation of information.
  2. Expense Tracking :
    • Add/Edit/Delete transactions
    • Categorize expenses (Food, Bills, etc.)
  3. Budget Management
    • Set monthly budget limits
    • Visual progress tracking
  4. Reporting
    • Interactive charts (React Charts.js)

Technologies used

  • Backend: Spring Boot, Spring Security, JPA/Hibernate
  • Frontend: React, Bootstrap
  • Database: PostgreSQL
  • Authentication: JWT

A user-friendly expense tracker with budget management, helping individuals monitor spending and maintain financial discipline.
