Portfolio Details

Educational web application for children

The application helps children discover and remember basic concepts (colors, animals, fruits, vegetables, jobs, parts of the body, numbers, clothing and shapes) through fun and colorful modules.

Available modules

    The application is divided into several educational modules :
    1. Color - Discovery of colors
    2. Animal - Learning Animal Names
    3. Fruits - Fruit Identification
    4. Vegetable - Getting to know vegetables
    5. Job - Exploration of professions
    6. Body - Knowledge of body parts
    7. Numbers - Introduction to numbers and simple calculations
    8. Clothe - Clothing recognition
    9. Shape - Learning shapes

Key Features

Each module has two parts:
  1. Learn part: To learn.
    • The child sees several images related to the theme of the module.
    • By clicking on an image, he hears the associated name in audio, which stimulates visual and auditory memory.
  2. Play Part: To play and test your knowledge.
    • For all modules (except Numbers):
      • An image appears randomly with four answer choices.
      • If the child chooses the correct answer, they hear a clapping sound and a success animation is displayed.
      • If answered incorrectly, a failure sound plays, and he can try again.
    • For the Numbers module :
      • An addition operation between two numbers (between 0 and 9) is generated.
      • The child types their answer in an input box, then clicks on the “equals” sign to validate.
      • If he answers correctly, he receives the same positive feedback as in the other modules; otherwise, a failure audio is played.

Navigation and ergonomics

For easy navigation, each module page includes:
  • An Exit button that returns to the home page.
  • A Play button to restart the game in the current module.
  • At the bottom of the page, a button with a green upward arrow also allows you to return to the home page with a single click.

Technologies used

  1. HTML :
    • Structure of pages and modules.
    • Organization of visual content for each part (Learn and Play).
    • Using audio tags to play naming, clapping, and error feedback sounds.
  2. CSS :
    • Visual styles of pages and interactive elements.
    • Use of bright colors and shapes to make the app attractive and child-friendly.
    • Animation of visual effects for correct answers, reinforcing the fun experience.
  3. JavaScript :
    • Interactive functionality of the application.
    • Handling click events to play sounds, check responses and trigger animations.
    • Dynamic generation of questions and answers in the Play section, including random display of choices.
    • Basic calculations for the Numbers module and validation of entered answers.

    4. Thanks to the use of these technologies, the application is accessible on most browsers and devices. Children can enjoy an interactive, colorful and engaging learning experience.
