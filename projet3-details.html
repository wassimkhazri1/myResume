Application Web de Chat en Temps Réel

Cette application de chat a été conçue pour offrir une plateforme simple, intuitive et efficace pour la communication en temps réel. Elle permet aux utilisateurs de s'inscrire, se connecter et rejoindre différentes salles de discussion afin d'interagir facilement entre eux.

Fonctionnalités Clés

  1. Authentification Sécurisée des Utilisateurs :
    • Inscription : Les nouveaux utilisateurs peuvent créer un compte en fournissant leurs informations.
    • Connexion : Les utilisateurs existants peuvent se connecter en toute sécurité avec leurs identifiants.
    • Mot de passe oublié : Récupération de compte pour les utilisateurs ayant oublié leur mot de passe.
  2. Salles de Chat en Temps Réel :
    • Plusieurs Salles de Discussion : L'application propose 3 salles de chat distinctes :
      • Salle 1 : Discussions générales.
      • Salle 2 : Sujets spécifiques.
      • Salle 3 : Conversations privées ou thématiques.
    • Messagerie Instantanée : Les messages sont envoyés et affichés en temps réel.
  3. Interface Utilisateur Conviviale :
    • Design réactif pour une expérience optimale sur ordinateur, tablette et mobile.
    • Navigation simple et intuitive grâce à Bootstrap pour le style.

Technologies Utilisées

  • Backend : PHP pour gérer la logique côté serveur et la gestion des utilisateurs.
  • Frontend : HTML, CSS et JavaScript pour créer une interface utilisateur interactive et réactive.
  • Frameworks : Bootstrap pour un design moderne et épuré.
  • Communication en Temps Réel : Utilisation de JavaScript pour la fonctionnalité de chat instantané.

Cette application de chat offre une solution légère et performante pour permettre aux utilisateurs de communiquer en temps réel, tout en garantissant un accès sécurisé et une expérience utilisateur agréable.
